搭建博客

搭建博客

  • 基于hexo

1.配环境

需要的资源：Node.js&GIT Win&Github

查看是否安装成功：

  • node -v
  • npm -v
  • git –version

2.准备SSH keys

将本地生成的ssh keys 在github的setting中关联

3.建立博客

1.本地文件中git bush进入界面

安装hexo npm install hexo-cli -g

2.hexo的初始化 hexo -init
3.生成 hexo g
4.本地测试hexo hexo s
5.启动服务 hexo d(也可hexo g -d一步到位)

  • 可能会输入相关账号密码或者密匙（网不好，建议用梯子)

  • 相关参考【2021最新版】保姆级Hexo+github搭建个人博客_哔哩哔哩_bilibili

4.选择主题

1.download theme

下载相关主题并在theme中替换

2.配置主题

在theme文件夹中的_config.yml中配置主题

  • 分类在发表文章后的 categories: 《类别》 # 分类进行标注
  • thumbnail: https://xxx.jpg # 略缩图
  • 以及注意背景图必须为jpg格式

5.评论区

主流的有

  • Github大礼包：gitment， gitalk（推荐），gitter（推荐）; 三个都支持Markdown；
  • 基于leancloud的无后端评论系统：Valine（推荐，支持Markdown）；
  • 国外的有几个：disqus(漂亮，但需翻墙)，hypercomments（推荐，不支持Markdown）.

本人选择了Valine

1.注册leanCloud

纪录

  • AppID，

  • AppKey

    在comment：valine：进行配置
